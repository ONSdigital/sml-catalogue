How do I submit new methods, or change existing methods, in the SML? (As an ONS Internal user)

Method requests can be raised to:

{{ onsList({ "variants": 'dashed', "itemsList": [ { "text": 'Have new methods developed for the Statistical Methods Library.' }, { "text": 'Add existing code to the Statistical Methods Library.' }, { "text": 'Change existing methods held within the Statistical Methods Library.' } ] }) }}

Method requests can be submitted to the ONS Statistical Methods Library via the IBM Business Automation Workflow (BAW). The BAW is used across the ONS for a variety of purposes, so you may have access to this tool already.

If you do not have access to the IBM Business Automation Workflow (BAW):

{{ onsList({ "element": 'ol', "itemsList": [ { "text": 'You can gain access to the BAW via ServiceNow in “I Want Something > 04 – Software, Applications & Platforms > BAW Support”.' }, { "text": 'Select the option “Add A New BAW User”.' }, { "text": 'Under “Process app name” enter “SML Method Request”.' }, { "text": 'Fill out the other information as appropriate to you.' } ] }) }}

If you already have access to the BAW, and want access to the SML forms:

{{ onsList({ "element": 'ol', "itemsList": [ { "text": 'You can gain access to the SML forms via ServiceNow in “I Want Something > 04 – Software, Applications & Platforms > BAW Support”.' }, { "text": 'Selection the option “Change A User’s Access To A Process App”.' }, { "text": 'Under “Process app name” enter “SML Method Request”.' }, { "text": 'Fill out the other information as appropriate to you.' } ] }) }}

Once you have been granted access to the SML forms on the BAW, login to the BAW here: {% from "components/external-link/_macro.njk" import onsExternalLink %} {{ onsExternalLink({ "url": 'https://www.bpm.ibmcloud.com/auth/index.jsp', "linkText": 'https://www.bpm.ibmcloud.com/auth/index.jsp' }) }}

Click on “Process Portal”. In the left-hand menu you will see “Create SML Method Request” - if you cannot see a menu on the left-hand side of the screen, click the three horizontal lines at the top left corner of the screen.

Click on "Create SML Method Request”, and fill out the information in this form to create the desired request.
